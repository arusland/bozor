# Bozor
Shopping list &amp; personal budget

## Install
* Configure `aplication.properties`
* `mvn package`
* Put war-file into any servlet container
* Or you can run `java -jar bozor-x-x.jar`

## Based on
* Spring Boot
* Thymeleaf
* AngularJS
* Twitter Bootstrap
* WebJars
* Mozilla Rhino
